Code

Code styling is kept basic – just wrap anything in a <code> and it will appear like this. For blocks of code, wrap a <code> with a <pre&t;.

.some-class {background-color: red; }
<pre><code>.some-class {
        background-color: red;
      }</code></pre>

      <!-- Remember every whitespace and break will be preserved in a <pre>, including indentation in your code -->
